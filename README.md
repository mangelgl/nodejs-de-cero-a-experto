## Dependencias

Para instalar las dependencias de un proyecto importado:
```
npm install
```
o su abreviatura:
```
npm i
```
Para instalar paquetes en modo global:
```
npm install <package>
```
Para instalar paquetes en modo desarrollo:
```
npm install -D <package>
```
